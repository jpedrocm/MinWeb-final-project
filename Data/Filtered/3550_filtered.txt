Back to Top The ASUS 64GB T100 Series Transformer Book is a tablet / notebook combo device that features a 10.1" HD 1366 x 768 multi-touch IPS display and a detachable keyboard dock with a touchpad for mouse-like control when using the tablet like a notebook. The tablet is equipped with a 64GB SSD, 2GB of DDR3 memory and a Windows 8.1 operating system that comes with a free copy of Microsoft Office 2013 Home and Student edition. It's powered by an Intel Baytrail-T Z3775 quad-core processor with a clock speed of 1.46 GHz. The processor also affords burst speeds up to 2.39 GHz for ultra-fast web browsing, quicker response time and more efficient multitasking performance. A front 1.2MP HD webcam allows you to video chat using the included Skype software. ASUS equipped the Transformer Book with 802.11 AGN (a/b/g/n) wireless connectivity and Bluetooth 4.0 wireless technology. It also features a micro-HDMI port, a micro-USB 2.0 port and a headphone/microphone combo audio jack. The Transformer Book supports expandable storage via the microSDXC memory card reader as well. Plus, you can use unlimited ASUS WebStorage free for a year. The Transformer Book's included keyboard dock connects to the tablet via Bluetooth 4.0 and it also has a USB 3.0 port for fast data transfer speeds.
